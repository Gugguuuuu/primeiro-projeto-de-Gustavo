HTML

1. O html é uma liguagem de marcação não de progamacão, é onde a pagina"web" é estruturada é como se o html fosse o esqueleto e as outras liguagens como js e css fossem a roupa que é colocada no html.

2. As tags mais importantes para o HTML geralmente são tag "html", "head", "body" que servem para estruturar o html, tambem as tags "h1", "p", "img", "a" que servem para textos, imagens, troca de paginas, titulo e etc... tambem temos tags para dividir o codigos que são as "div".Podemos classcificar todas as tags usando "class="" " e "id="" ".
